# iron-chef
Java application for managing kitchen inventory, ingredients and expiration dates.

## Current Features
- Add an ingredient name
- Enter expiration month
- Enter expiration year
- Store ingredient in live inventory
- Sort inventory in ascending/descending order
- Clear input fields after adding an ingredient

## Technologies Used
- Java
- Java Swing
- ArrayList

## Project Status
This project is currently in early development. The current version focuses on building the core GUI and basic ingredient entry functionality.

## Planned Features
- More attractive GUI
- Validate user input
- Remove ingredients from inventory
- Search for ingredients
- Connect ingredients to recipe management features

## How to Run
1. Open project in Eclipse
2. Run "IronChefApp.java"
3. Enter an ingredient name and expiration information
4. Click **Add** button
